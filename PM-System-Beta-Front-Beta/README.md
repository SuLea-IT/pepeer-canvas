### Backend Deployment

1. Clone the project repository

   ```cmd
    git clone https://github.com/SuLea-IT/PM-System-Beta-Front.git
   ```

2. Navigate to the project directory

   ```cmd
   cd PM-System-Beta-Front
   ```

3. Install dependencies

   ```
   npm install
   ```

4. Run the project

   ```cmd
   npm run dev
   ```
